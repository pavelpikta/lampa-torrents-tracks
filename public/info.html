FFprobe Tracks Analyzer

Your own self-hosted media analysis tool

Modern Web Interface

Beautiful Tailwind CSS design with responsive layout. Dark theme optimized for comfortable viewing.

TorrServer API

Direct integration with TorrServer's FFprobe API. Fast and reliable media analysis.

HTTP API

Built-in Node.js HTTP API server. RESTful endpoint with secure proxying to TorrServer. HTTPS ready via cloudflared.

Easy Deployment

Multiple deployment options: Docker, VPS, static hosting. Comprehensive guides included.
